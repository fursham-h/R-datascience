Cheat sheet 

- list of commands we've learned so far

`pwd` print working directory
`cd` change directory (e.g., `cd ~/Desktop` to go to Desktop folder in your home directory)
`ls` list content of directory (e.g., `ls -l Downloads` to list content of folder [Downloads] in long format)
`man [command]` or `[command] --help` get help (e.g., `man ls` or `ls --help` to get help on [ls] command)
`mkdir` make new directory (e.g., `mkdir RNAseq_workshop` to create new folder [RNAseq_workshop])
`nano` open text editor (e.g., `nano test.txt` to create a new text file called [test.txt] in the current folder)
`mv` move or rename (e.g., `mv test.txt test_new.txt` to rename the file [test.txt] to [test_new.txt] in the current folder)
`cp` copy (e.g., `cp test_new.txt RNAseq_workshop/draft_new.txt` to copy the file [test_new.txt] to a folder called [RNAseq_workshop])
`rm` remove (e.g., `rm test_new.txt` to remove the file [test_new.txt] from the current folder)
`wc` word count (e.g., `wc -l file.txt` to count the lines in the file [file.txt])
`cat` print to screen (Be aware that cat will print out the entire content, even if it is a big file)
`less` to explore the content of a file (arrows up/down to scroll, q to exit)
`head` to print to screen the beginning of a file (e.g., `head -n 2 file.txt` print the first 2 lines of the file)
`tail` to print to screen the end of a file (e.g., `tail -n 2 file.txt` print the last 2 lines of the file)
`grep` to find a matching pattern in a file (e.g., `grep NN SRR7311317_1.fastq` find the N doublets in the RNA-seq file [ SRR7311317_1.fastq])
`>` or `>>` to redirect the output of a command to a file (either overwrite or append) (e.g., `grep NN SRR7311317_1.fastq > Ns.txt` write in a file called [SRR7311317_1.fastq] the output of the grep search)
`|` pipe, direct the output of a command as input to the next (e.g., `grep NN SRR7311317_1.fastq | wc -l` search the file [SRR7311317_1.fastq] for N doublets and then count the number of lines)
`gunzip` and `gzip` to decompress and compress .gz files respectively (e.g., `gunzip SRR7311317_1.fastq.gz` to unzip the file [SRR7311317_1.fastq.gz] and `gzip SRR7311317_1.fastq` to compress the file [SRR7311317_1.fastq])
`curl` to download files from the internet. (e.g., `curl -O ftp://ftp.sra.ebi.ac.uk/vol1/fastq/SRR731/007/SRR7311317/SRR7311317_1.fastq.gz` to download the FASTQ file to the current working directory as the original name of the file [SRR7311317_1.fastq.gz])

- special hotkeys/characters we've learned so far
`Ctrl+z` or `Command+z` to force quit a running command. Useful when you accidentally run `cat` on a large file.
`~` shortcut to the home directory
`*` wildcards for any number of characters or one single character (e.g., `ls -l /bin/b*` to list all files beginning with 'b' in the /bin/ directory)
